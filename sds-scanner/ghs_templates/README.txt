GHS Reference Template Images
==============================

Place the 9 GHS pictogram PNG files here to enable full pictogram identification.
Without these files the app runs in "shape detection only" mode — it still finds
the diamond shapes but cannot name which symbol is inside.

Required filenames (lowercase):
  ghs01.png  — Exploding Bomb
  ghs02.png  — Flame
  ghs03.png  — Flame Over Circle (Oxidizer)
  ghs04.png  — Compressed Gas
  ghs05.png  — Corrosion
  ghs06.png  — Skull and Crossbones (Toxic)
  ghs07.png  — Exclamation Mark (Harmful/Irritant)
  ghs08.png  — Health Hazard (Serious)
  ghs09.png  — Environmental Hazard

Download from:
  https://unece.org/transport/dangerous-goods/ghs-pictograms
  (Download each symbol image and rename to ghs01.png ... ghs09.png)

TIP: Use the inner symbol image only (without the diamond border) for best
template matching accuracy. The border is already detected by color analysis.
